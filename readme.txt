Display Knowledge Panel information using a shortcode.

== Description ==
Knowledge Panel Viewer is a WordPress plugin that allows you to easily display information from the Google Knowledge Graph using a simple shortcode. This is particularly useful for showcasing details about entities, people, or topics.

= Features =
- Use the [knowledge_panel_viewer] shortcode to embed the Knowledge Panel Viewer form.
- Input a query and result limit to fetch and display Knowledge Panel information.
- Instructions available in the WordPress dashboard for easy usage.

== Installation ==
1. Upload the entire `knowledge-panel-viewer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the [knowledge_panel_viewer] shortcode in any post or page to display the Knowledge Panel Viewer form.

== Frequently Asked Questions ==
= How do I use the Knowledge Panel Viewer? =
- Simply insert the [knowledge_panel_viewer] shortcode into any post or page. Fill in the query and result limit, then click the "Search" button to view Knowledge Panel information.

= Can I customize the appearance of the form? =
- You can customize the styles by modifying the CSS in your theme or using a custom stylesheet.
